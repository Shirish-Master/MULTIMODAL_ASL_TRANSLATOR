1. Dataset Setup

To use the ASL WLASL Converter, you need to set up the WLASL dataset:

  1. Download the WLASL_v0.3.json metadata file
  2. Download the video files
  3. Place files in the correct directories
The WLASL dataset is required for both text-to-video and video-to-text conversion.
WLASL Repository
2. Model Setup

For video-to-text recognition, you need the pretrained I3D model:

  1. Download the pretrained I3D model weights
  2. Place the model file in the models directory
The pretrained model is required for video-to-text recognition.
